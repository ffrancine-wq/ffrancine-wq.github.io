# ForroRaiz-CantoresEBandasclassicos
o projeto tem como objetivo apresentar, de forma interativa, cantores e bandas clássicos do forró, valorizando a cultura nordestina através da tecnologia.
O site permite que o usuário selecione preferências musicais, como tipo de gravação, instrumento principal, estilo de forró e década de maior destaque. Com base nessas escolhas, serão apresentados artistas que mais se relacionam com o perfil selecionado. 
E também serão mostradas algumas caracteristicas introdutorias sobre os artistas como estilo musical, maior sucesso, inicio da carreira, músicas publicadas.
Além disso o projeto contará com uma breve introdução ao gênero.
